## Python web on vercel and Flask

<https://github.com/vercel/examples/tree/main/python/flask2>

## run flask
```bash
export FLASK_APP=app.py
export FLASK_ENV=development
```
```bash
flask run
```

## run local

```bash
npm i -g vercel
vercel dev

```
